General notes

Please note, the following list refers to the default settings, i.e. when the left mouse button is used for drawing and selection, while the right mouse button for panning the image left/right and up/down. The buttons may be swapped using the Preference dialog or by pressing the Swap left and right mouse buttons on the toolbar.

The default behaviour of the mouse wheel is to change the slices and the Q, W keyboard keys to zoom in/zoom out; this may be swapped using the mouse key in the toolbar or using the Preferences dialog.

The key shortcuts can be customized in the File->Preferences->Shortcuts dialog.

Combination of mouse and keys

^ Ctrl+mouse wheel change size of the brush and other selection tools
^ Ctrl+⇧ Shift+mouse wheel change size of the brush and other selection tools in bigger increments

Interaction with ROIs

To interact with ROIs (for example to change its size or position) please use right click over the ROI name in the ROI List of the ROI Panel to start a popup menu. Choose Edit to modify the selected ROI.

When the Edit mode is enabled the following actions are available:

  • left click+drag to change position of the selected ROI
  • right click on a vertex of the selected ROI to delete it
  • hold A+left click on an edge of the selected ROI to add a new vertex
  • double click to accept changes
