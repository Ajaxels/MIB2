Did you know...

The 'D' checkbox in the Segmentation panel allows you to specify segmentation tools that can be quickly accessed using the 'D' key shortcut.
  • Select two or more frequently used segmentation tools.
  • Ensure that each selected tool has the D checkbox checked.
  • Press the D key to toggle between these tools.
Additionally, you can use the Shift+D and Ctrl+D shortcuts to define and access two favorite tools. These can be selected from
Menu → File → Preferences → Segmentation Tools.
