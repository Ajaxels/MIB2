Did you know...

Most segmentation tools in the Segmentation panel allow users to save their settings as presets. Each tool supports up to three presets.

To store the configuration of the selected tool, press Shift+1, Shift+2, or Shift+3 to save it as preset 1, 2, or 3, respectively.
To restore a saved preset, press 1, 2, or 3.

These key shortcuts can be customized in
Menu → File → Preferences → Keyboard Shortcuts
